【论文阅读】RGB-D Scene Recognition with Object-to-Object Relation

· 2019-05-28 ·

场景识别存在的问题

  1. 场景是一个抽象的概念，而它又通常是由一些具体的物体或概念构成的
  2. 从一些low-level的视觉特征进行抽象概念的推理存在语义鸿沟
  3. 物体的共现不能很好的解决场景歧义的问题

目前的解决方案

  1. 通过构建一个“bag model”，利用low-level的视觉特征进行推理识别
  2. Two-steps：1）构建中间特征对图像进行表示 2）利用中间特征进行推理识别
  3. 利用空间信息对场景进行表示，解决仅利用中间特征带来的歧义问题

论文提出的解决方案

  1. 利用RGB图像信息和depth data进行图像目标检测，得到检测物体的bounding boxes
  2. 将得到的候选框合并
  3. 然后利用识别结果构建object-to-object relation（OOR）表示
